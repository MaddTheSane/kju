System Specifications

Host

To run Q on Your Computer, the Computer must meet the following specifications:
  • G4/G5 Processor or Intel Core Solo Processor
  • 256 MB RAM
  • OS X 10.3.x/10.4.x

Guest

For system emulation, the following hardware targets are supported:
  • PC (x86 or x86_64 processor)
  • PREP (PowerPC processor)
  • G3 BW PowerMac (PowerPC processor)
  • Mac99 PowerMac (PowerPC processor, in progress)
  • Sun4m (32-bit Sparc processor)
  • Sun4u (64-bit Sparc processor, in progress)
  • Malta board (32-bit MIPS processor, in progress)
The QEMU System emulator simulates the following PC peripherals:
  • i440FX host PCI bridge and PIIX3 PCI to ISA bridge
  • Graphiccard
    • Cirrus CLGD 5446 PCI VGA card
    • dummy VGA card with Bochs VESA extensions (hardware level, including all non standard modes).
  • PS/2 mouse and keyboard
  • 2 PCI IDE interfaces with hard disk and CD-ROM support
  • Floppy disk
  • NE2000 PCI network adapters
  • Serial ports
  • Soundcard
    • Adlib card
    • Soundblaster 16 card
    • ES1370 card
