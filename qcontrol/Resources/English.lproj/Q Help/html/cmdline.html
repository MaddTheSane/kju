Starting Q from the Commadline

Q can be started from the terminal or from scripts.
The executable is located at Q.app/Contents/MacOS/qemu.

Q expands the Command Line Arguments of QEMU by:
  • -cocoaname NAME
    this name will be displayed at the top of the Window. It is used to save the VM or the Thumbnails.
  • -cocoapath FULLPATH
    Is used to save the VM or the Thumbnails.
  • -cocoalivethumbnail
    Toggles Thumbnails to ON.
  • -cocoanodialogs
    Toggles Savedialogs to OFF. You wont be able to save a VM!
  • -cocoaopengl
    Use OpenGL to display the guest PC.
