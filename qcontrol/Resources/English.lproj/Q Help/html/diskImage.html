Create a new Diskimage

To create a new Diskimage select "create new Diskimage..." from the Menu "Tools".

  1. Select the Size Please enter the size of the Diskimage. The size is messured in Megabites. See for size recommandations.

  2. Select the Format
    • qcow
      qcow is QEMUs own format for diskimages. qcow images start with a few kilobites and grow to the desired size. This saves Harddiskspace until the images eventually reaches the maximum size. qcow Images do not shrink, when you delete data on them.
    • raw
      raw, also known as img or dsk are just plain diskimages, with not compression. They take longer to create and have all the time theyr original size. This imageformat can be mounted by OS X (you may have to rename them to .img). Please remember, that OS X will write special files to the image, if you mount it under OS X.


  3. Select a Filename Please enter a Location and Filename for your new Diskimage. Please remember the place of your new Diskimage.

  4. Please wait...
    qcow images are created almost instanteley. raw images can take some minutes to create! (depending oon the size).
  5. Your Done!
