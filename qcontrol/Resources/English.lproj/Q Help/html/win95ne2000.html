Win95: Installing Networkcard

If You don't need the advanced features of the later Windows Versions, Windows 95 is the fastest choice. Unfortunately, Windows 95 does not recognise the Networkcard during Installation. But You can select the corresponding Driver manually:
  1. open "Systemproperties", select "System"
  2. select TAB "Devices", select "PCI Ethernet Controller"
  3. select "Driver", select "update Driver"
  4. select "Yes recommended"
  5. select "back"!
  6. select "Netwotk Adaptor"
  7. select "Novel/Anthem", select "NE2000 compatible"
  8. enter Name, Workgroup and Description
  9. close dialog, reboot
