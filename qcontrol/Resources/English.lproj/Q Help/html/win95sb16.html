Win95: Installing Soundblaster16

If You don't need the advanced features of the later Windows Versions, Windows 95 is the fastest choice. Unfortunately, Windows 95 does not recognise the Soundblaster16 during Installation. But You can select the corresponding Driver manually:
  1. open "Systemproperties", select "Hardware"
  2. select "continue"
  3. don't search for new Harware and continue
  4. select "Audio- Video- and Gamecontroller"
  5. select "Creative Labs", select "Creative Labs Sound Blaster 16 or AWE-32" (not plug'n'play!)
  6. complete the installation.
To get rid of the nasty new harware found nag:
  1. open "Systemproperties", select "System"
  2. select Tab "Devices"
  3. doubleclick "PCI Multimedia Audio Device"
  4. select "disable device..."
